Projects

  • NOPTREX Readout Software

    Presenting at NCUR 2023 and APS April Meeting 2023.

    A multi-channel readout software for the Neutron Optics Parity and Time-Reversal Experiment (NOPTREX). The experiment is investigating time-reversal violations in low-energy neutron reactions by monitoring the levels of gamma radiation emitted after the collision of an accelerated neutron with a heavy target. Through this, we hope to be able to explain the Baryon Asymmetry of the early universe.

    This readout software was developed in C++ using the ROOT framework and communicated in real time with custom firmware on a CAEN DT5560SE digitizer, processing and outputting its data. It was successfully tested on a J-PARC particle accelerator.

  • Hydraulic Erosion Simulator

    This project allows the user to load in terrain measurements or procedurally generate random terrain, then use a hydraulic erosion algorithm on that terrain. This can either be used to automate the creation of realistic random terrain or simulate actual erosion on measured terrain.

    Built for University of Kentucky's Honors Algorithm and Design Final Project, Spring 2022.

    Press H to hide all UI elements and G to hide all tips.

  • Phantom Mansion - MLH Hackathon 1st Place Winner

    A haunted house roguelike where your goal is to find treasure at the top of the house. Multiple difficulty levels are available and high scores are saved. All levels are randomly generated using graph traversal algorithms.

    Won first place at Cathacks VIII Hackathon. Presented at University of Kentucky E-Day 2023.

    To input a control, type it into the console and press "Enter." Look at the help menu to view the rest of the controls.

  • Platformer

  • Machine Learning Calculator

    This is a handwriting-recognizing calculator I created. You're able to draw on a canvas and a machine learning algorithm will try to determine what number you've drawn by the pixels filled in on the canvas. You're also able to change some of the settings and retrain the algorithm if you want to change how it works.

    This uses the MNIST handwritten digit database. Press escape to close the program and backspace to remove the last registered number.

  • File Transfer Protocol

    This is a set of 3 client-server file transfer protocols. They are all built on the same base, with slight differences in how the different machines communicate. This also contains some text files to test the protocols.

    Users can request and download files contained on the server or the other client. Clients never communicate directly with each other, only indirectly through the server which can make requests on behalf of the clients.

  • Pixel Physics

    This is a small game/proof of concept that physically simulates 1,500 pixels across 5 types of materials. Each of the materials acts differently than one another which allows the user to create a large number of unique scenarios. Press the left mouse button to place material and the right mouse button to remove it. Press R to reset the canvas and escape to close the program.

    Controls:
    1- Sand
    2- Wood
    3- Water
    4- Fire
    5- Stone

  • Rope Physics

    This is a rope simulation game where your goal is to keep the white ball inside the bounds of the screen. You can do this by changing the start and end points of the rope and using it to push the ball in certain directions. The rope will react to the ball and move realistically without a Unity collider or hinge component. Press the left mouse button to change the position of the blue end of the rope and press the right mouse button to change the red end of the rope. Press R to reset the game and escape to close the program.
